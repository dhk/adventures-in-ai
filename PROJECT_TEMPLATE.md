# Project Template
Use this template when adding a new project.

## Project Summary
- Status: Idea | Exploring | Prototype | Draft | Dataset | Published | Broken
- Problem: What question or problem this project tackles
- Why AI: Why AI is the right tool (or why testing it is useful)
- Artifacts: Key files or links (`README`, notebooks, data, code)

## Background
One paragraph explaining the motivation and context.

## What I did
Short bullet list of the core steps or experiments.

## What I learned
2–5 concrete takeaways, framed as decisions or insights.

## Notes / To-Dos
- [ ] Next steps or open questions
