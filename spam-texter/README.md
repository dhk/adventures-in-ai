# About spam-texter
## Project Summary
- Status: Idea
- Problem: Respond to spam texts automatically and keep them engaged
- Why AI: Generate varied, long-running conversational replies
- Artifacts: Notes and early planning

My glorious vision is that when spambots reach out to me via text, my computer responds.

Things I'm thinking about:
1. Teaching it to learn, to keep the spambots engaged for as long as possible
1. Having different personae:
  - "Unhinged" is a good one
  - Little old lady in Springfield, OH
  - A Nigerian spammer (yup — see if we can't convince them they're talking to one another)
  - ...
1. Somehow hook it up to my phone so it can take over the appropriate conversations

You get the idea.

## How to do this
I don't really know. But my partner in crime is ChatGPT. I'll post bits of the transcript along the way so that you can see how I'm doing it.
