# Resume-Parser
Parsing your resume to generate a word cloud for your Linkedin header

## V1 / V2 / V3 wordcloud comparison

This lab compares three approaches to the same LinkedIn header wordcloud:

| Version | Script | Notes |
|--------|--------|--------|
| V1 | `v1_original.py` | Baseline |
| V2 | `v2_claude_native.py` | With `v2_demo_weights.py` for weighting demo |
| V3 | `v3_claude_code.py` | Claude Code–driven variant |

Rendered outputs: `output_v1.png`, `output_v2.png`, `output_v3.png`. Header asset: `linkedin_wordcloud_header.png`.

**Source branch:** `wordcloud-comparisons` (cherry-picked from prior `feat/podcast-panel` work, merged to `main`).
