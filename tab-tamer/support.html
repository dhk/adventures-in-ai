Tab Tamer
by DHK

A Chrome extension for managing large numbers of open tabs. Sort, search, group, and close — right from your toolbar.

Features

Search

Filter your tabs in real time by title or URL. Results update as you type, with a one-click clear button.

Sort

Sort tabs by title (A→Z or Z→A) or by last visited (newest or oldest first). Sorting physically reorders tabs in Chrome's tab bar.

Group by Site

Instantly group all tabs by domain using Chrome's native tab groups — color-coded in the tab bar and collapsible in the popup.

Ungroup

Disband all tab groups with one click, returning tabs to a flat ungrouped list in the tab bar.

Multi-select & Close

Check any number of tabs and close them all at once with the "Close (N)" button. Select All / Deselect All included.

Switch Tabs

Click any tab row (not the checkbox) to jump straight to that tab. The popup closes automatically.

How to Install

Tab Tamer is loaded directly from a local folder — no Chrome Web Store required.

Step 1 — Get the files

Download the tab-tamer.zip from the adventures-in-ai repository and unzip it, or clone the repo:

git clone https://github.com/dhk/adventures-in-ai.git

Step 2 — Load in Chrome

  1. Open Chrome and go to chrome://extensions
  2. Toggle Developer mode on (top-right corner)
  3. Click Load unpacked
  4. Select the tab-tamer/ folder
  5. The Tab Tamer icon appears in your toolbar — click to open

Don't see the icon? Click the puzzle-piece icon in the Chrome toolbar and pin Tab Tamer.

Updating

After pulling new changes, go to chrome://extensions, find Tab Tamer, and click the ↺ refresh icon.

How to Use

Search

Type in the search bar at the top to filter tabs by title or URL. The list updates instantly. Hit the ✕ button or clear the text to show all tabs again.

Sort

Click Title to sort alphabetically. Click again to reverse. Click Last Visited to put the most recently accessed tab first. Click again for oldest-first. Sorting reorders your actual Chrome tabs, not just the popup view.

Group by Site

Click Group by Site to create a native Chrome tab group for each domain. Groups are color-coded and visible in the tab bar. In the popup, each group is a collapsible section — click the group header to collapse or expand it.

Ungroup All

Click Ungroup All to disband every group and return to a flat tab list.

Close Tabs

Check the box next to any tabs you want to close. The Close (N) button appears — click it to close all selected tabs at once. Use Select All to check every visible tab.
