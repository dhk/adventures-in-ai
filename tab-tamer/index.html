Chrome Extension

Tab Tamer
by DHK

Sort, search, group, and close your open Chrome tabs — right from the toolbar. No clutter, no new tabs, no backend.

⬇ Add to Chrome — It's free View on GitHub

Features

Search

Filter tabs in real time by title or URL. Results update as you type.

Sort

Sort by title or last visited. Sorting physically reorders tabs in Chrome's tab bar.

Group by Site

One click creates native Chrome tab groups per domain, color-coded in the tab bar.

Ungroup

Disband all tab groups instantly and return to a flat list.

Multi-select & Close

Check any tabs you don't need and close them all at once.

Privacy-first

No page content is ever read. No data leaves your browser. Ever.

Install

Chrome Web Store Recommended

One click, automatic updates, no setup required.

⬇ Add to Chrome

From GitHub

Load the latest source directly in developer mode.

git clone https://github.com/dhk/adventures-in-ai.git
  1. Open Chrome and go to chrome://extensions
  2. Toggle Developer mode on (top-right)
  3. Click Load unpacked
  4. Select the tab-tamer/ folder inside the cloned repo
  5. Pin the Tab Tamer icon from the puzzle-piece menu

To update: git pull, then hit ↺ on chrome://extensions.
