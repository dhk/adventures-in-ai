# LinkedIn Search URL Generator
## Project Summary
- Status: Prototype
- Problem: Create quick LinkedIn search URLs for two people
- Why AI: Rapidly generate a lightweight, static UI
- Artifacts: `index.html`

This is a single-page HTML tool that accepts two names and generates LinkedIn search URLs restricted to first-degree connections.

## How to run
Open `index.html` in a browser.
