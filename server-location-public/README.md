# Experiments in AI: Google-worthy code
## Project Summary
- Status: Published
- Problem: Package and ship a Chrome extension built with AI assistance
- Why AI: Move from prototype to store-ready release quickly
- Artifacts: Extension package, screenshots, store listing

The original project is [found here](../server-location/README.md) and offers a browser-bar popup showing you the server location of a page you're currently on.

I decided to go full-circle; and see if I could get this code published as a Chrome Web Store extension

![image](https://github.com/user-attachments/assets/68014592-2c87-4a76-b229-43c941511f9b)


## Caveats
* I had to use [imresizer.com](https://imresizer.com/) to create correctly sized icons.
* I had to ask Claude to change the manifest version from V2 to V3. This is probably because the version changed since Claude last learned.
* After a couple of iterations with the review team, the app was accepted for publication and is available in the Chrome Web Store.

## Where to find it
The app is available in the [Chrome Web Store](https://chromewebstore.google.com/detail/server-location-finder/makphdkahgmkhdjjicjngcoghbclnmom).
