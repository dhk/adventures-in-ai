# Number Finder (Pi Sequence)
## Project Summary
- Status: Prototype
- Problem: Find where a digit sequence appears in pi
- Why AI: Quick scaffolding for a tiny Flask demo
- Artifacts: `number-finder.py`, `index.html`, `requirements.txt`

This Flask app searches the first 1,000,000 digits of pi for a user-provided sequence (up to 9 digits).

## How to run
1. Install dependencies: `pip install -r requirements.txt`
2. Run: `python number-finder.py`
3. Open `http://localhost:5000`
