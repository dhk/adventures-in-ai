# adventures-in-ai
Practical experiments and notes on how AI tooling changes analytics work, training, and product discovery.

This repo is for analysts, data-minded PMs, and builders who want concrete examples, not just opinions. Each project pairs a real question with code, prompts, and takeaways.

## What this repo explores
- how AI-assisted tools support analytics exploration
- when and how to incorporate AI into analyst workflows
- ways AI can accelerate training and skill-building
- how AI shifts product discovery and definition

Along the way I include working code examples, because argument without evidence veers toward opinion.

## How to use this repo
- Start with a project that matches your curiosity.
- Skim the README for context, then dive into code or notes.
- Use the prompts and examples as a starting point, not a prescription.
- New projects should follow the [`PROJECT_TEMPLATE.md`](PROJECT_TEMPLATE.md).

## Cross-project lessons learned
- LLMs are strong at ideation, summarization, and scaffolding, but weak at precision.
- Iteration speed matters more than perfect initial prompts.
- AI-assisted prototyping changes discovery: you can show working artifacts sooner.
- The easiest wins are in "analysis + narrative" workflows, not full automation.
- Documentation and transcripts are part of the artifact, not just the build.
