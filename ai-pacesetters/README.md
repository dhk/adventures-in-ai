# Introduction
## Project Summary
- Status: Dataset
- Problem: Summarize how AI is used in performance management by pacesetter orgs
- Why AI: Structure and compare qualitative descriptions across organizations
- Artifacts: `ai-pacesetters.md/.csv`, `ai-pacesetters-with-outstrip-factor.md/.csv`
