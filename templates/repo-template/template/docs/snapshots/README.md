# Snapshots

Dated, frozen records — `YYYY-MM-DD-<topic>.md`. Write-once: the output of
a design sprint, the reasoning behind a pivot, anything worth keeping
permanently as-is.

Not for session status — that's `HANDOFF.md` at the repo root, which stays
current instead of accumulating. If you're tempted to add "update" entries
to a snapshot, you actually want `HANDOFF.md`.
